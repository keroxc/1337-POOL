# Pool42 Project Overview
## A curated archive of my journey through the 42 School Piscine вЂ” a rigorous, peer-to-peer intensive program that challenges students to master the fundamentals of software engineering through hands-on C programming, shell scripting, and algorithmic problem solving.

This repository documents my progression from foundational concepts to advanced system-level programming, completed entirely through self-directed learning, peer collaboration, and relentless debugging. Each project reflects the 42 methodology: no traditional teachers, no lectures вЂ” just code, man pages, and the collective intelligence of the community.

Key Areas Covered:

Shell & System вЂ” Unix fundamentals, bash scripting, process management

C Programming вЂ” Memory management, pointers, data structures, algorithms

Problem Solving вЂ” Logical reasoning, optimization, and debugging discipline

Collaboration вЂ” Peer reviews, group projects, and knowledge sharing

These projects represent not just code, but a transformation in how I approach problems, debug systems, and learn independently. The 42 Piscine taught me that the best engineer isn't the one who knows all the answers, but the one who isn't afraid to dive into the unknown and emerge with understanding.
